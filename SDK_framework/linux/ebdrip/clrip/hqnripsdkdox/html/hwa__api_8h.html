HWA RDR Types.

RDR Types for the RDR_CLASS_HWA Class defined in rdrapi.h.

In all cases the RDR ID is the bit depth - 1, 2 or 4. It is permissible to register the same table for different bit depths.

In the case of RDR_HWA_GIPA2_TONERLIMIT only the RDR length is used, the ptr is ignored. The length gives the actual toner limit in the range 255..1023.

Enumerator:
RDR_HWA_GIPA2_CMM 

The 3D LUT for CMM, in hwa_gipa2_lut format.

RDR_HWA_GIPA2_BLACK
