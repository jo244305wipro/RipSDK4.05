Detailed Description

Format of RDR_HWA_GIPA2_USER table.

This points to the User Gamma tables used by GIPA2.

Field Documentation

uint32 flags

Options: b0 clear: Tables must be copied into the buffer. set: HWA can address the tables directly.

size_t length

Size of the above tables - should be 1024.

uint8* user

Four 256 byte tables for Cyan, Magenta, Yellow and Black making 1024 bytes in total, 8 byte aligned.
